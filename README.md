# Iris-Titanic

This repository contains **practical Jupyter notebook explorations of Kaggle datasets**. 

## Contents

- `iris.ipynb`  
  Exploratory analysis of the classic **Iris dataset**, focusing on feature relationships, class separability, and basic statistical insights.

- `titanic.ipynb`  
  Exploratory analysis of the **Titanic dataset**, examining survival patterns based on passenger attributes such as age, gender, class, and fare.

## Purpose

These notebooks are intended as:

- A **personal learning notebook**
- A reference for common EDA workflows
- Hands-on practice working with Kaggle datasets

## Notes

- Datasets used are publicly available on Kaggle.
- These notebooks are exploratory in nature and may include iterative experimentation.
- Code and analysis are intentionally kept readable and educational.
